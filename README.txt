CMPT 459 Project - README File

Inside this code.zip package includes the following program:
timeseries.py - plot global and country cases; get the top 10 countries with the most confirmed cases
forecast.py - predict the cases of the ten countries

Please ENSURE the fbprophet package is installed into the computer before running forecast.py,
as well as pystan, which is requried by fbprophet to run a module called "convertdate"
Otherwise, it will report an error saying the module does not exist

The three datasets used for the projects are the confirmed cases, deaths and recoveries of COVID-19 
of 188 countries from the date Jan 22, 2020 - Jan 8, 2020.
The dataset used for this project are also included in this package. However, the program
uses the repository links to retrieve the dataset.
